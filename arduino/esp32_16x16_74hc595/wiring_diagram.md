| GND       | Pin 4 (OE)   | 输出使能 | 黑色 |

## 注意事项

### ⚠️ 1. 确认引脚定义
**不同厂家的 16 针接口定义可能不同！**

请先确认你的点阵屏引脚：
- 查看模块背面丝印
- 查看卖家提供的资料
- 或用万用表测量（VCC/GND 通常连通多个引脚）

### ⚠️ 2. 电源
- 大部分模块用 **3.3V** 供电
- 少数用 **5V**，请确认后再通电
- 接错可能烧毁模块！

### ⚠️ 3. 如果显示异常
- 检查 DATA/CLK/LATCH 是否接反
- 尝试交换行/列顺序
- 调整 `BRIGHTNESS_DELAY` 改变亮度

## 调试方法

### 串口监视器
上传代码后打开 Arduino IDE 的串口监视器（115200 波特率）：

```
ESP32 16x16 Dot Matrix (74HC595) Starting...
Setup complete!
Displaying numbers 0-9
Displaying letters A-F
Drawing heart
Scrolling text
Animating
```

### 如果点阵屏不亮
1. 检查电源接线（VCC/GND）
2. 检查 OE 引脚是否接地
3. 用万用表测供电电压

### 如果显示乱码
1. 检查 DATA/CLK/LATCH 顺序
2. 确认是共阴还是共阳（代码里调整 `rowSelect` 逻辑）
3. 尝试交换字节顺序

## 代码引脚修改

如果实际接线用了不同引脚，修改代码顶部：

```cpp
#define DATA_PIN    23    // 改成你的 DATA 引脚
#define CLK_PIN     18    // 改成你的 CLK 引脚
#define LATCH_PIN   5     // 改成你的 LATCH 引脚
```
